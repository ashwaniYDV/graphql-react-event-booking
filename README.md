# graphql-react-event-booking

> GraphQL, React

## Quick Start

```bash
# Install dependencies (server & client)
npm run add

# Run server & client (:4500 & :3000)
npm run dev

# Server only (:4500)
npm run server

# Client only (:3000)
npm run client

# Graphiql - http://localhost:4500/graphql
```
